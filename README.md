# Bootstrap

Some bootstrap 4 exercises

## Run Locally

Clone the project

```bash
git clone https://github.com/isaaccolls/bootstrap.git
```

Go to the project directory

```bash
cd bootstrap
```

Start the some server like

```bash
python3 -m http.server 8000
```
